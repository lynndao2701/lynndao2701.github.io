## Overview

Property ownership transfers traditionally rely on intermediaries, manual processes, and fragmented record-keeping — creating inefficiencies, transparency issues, and security risks. This project designed and implemented a blockchain-based escrow system to automate ownership transfers, reduce reliance on intermediaries, and strengthen data governance and auditability throughout the process.

Completed for **BUSA3007 – Data Governance Using Blockchain for Business** at the University of Newcastle.

## What I Did

- Designed a **decentralised escrow architecture** using Ethereum smart contracts (Solidity) to automate the property transfer process
- Implemented **NFT-based digital property ownership** to represent and transfer real estate titles on-chain, ensuring immutable and transparent records
- Built a **ReactJS + Web3.js decentralised application (dApp)** integrated with MetaMask for user authentication and transaction signing
- Managed property and user data through **MongoDB** as the off-chain data layer
- Tested the proof-of-concept using **Hardhat** and **Ganache** local blockchain environments
- Applied data governance principles to ensure security, transparency, and auditability throughout the transfer workflow

## Outcomes

- Solidity smart contracts for automated escrow and ownership transfer
- NFT-based digital property ownership model
- ReactJS + Web3.js decentralised web application
- MetaMask-integrated transaction workflow
- Fully tested proof-of-concept via Hardhat and Ganache

## Tools & Techniques

Solidity, Ethereum, Smart Contracts, NFT, ReactJS, Web3.js, MetaMask, MongoDB, Hardhat, Ganache, data governance, blockchain architecture, decentralised application (dApp) development

## Outcome

The project demonstrated how blockchain technology can address key pain points in property transfer — reducing intermediary dependency, improving transparency, and strengthening data governance — with practical implications for real estate, legal, and financial services.
