## Overview

Wildlife-vehicle collisions are a growing road safety and conservation problem near forested roads. This project develops an AI-powered detection system that identifies animals near roads in real time, evaluates collision risk, and triggers alerts to drivers — optimised for edge deployment to support low-latency decision-making in real-world transport environments.

Completed for **COMP3330** at the University of Newcastle.

## What I Did

- Designed a **multi-agent system architecture** with specialised agents handling detection, risk assessment, communication, and monitoring
- Trained **YOLOv8x** for real-time wildlife object detection across multiple animal classes, benchmarked against YOLOv8n and Faster R-CNN
- Applied **data augmentation** to improve model robustness across varying environmental conditions including day, night, rain, and fog
- Evaluated detection performance and system response reliability across conditions
- Assessed deployment feasibility for edge AI and roadside hardware scenarios
- Conducted risk analysis and aligned the solution with UN SDG 11 (Sustainable Communities)

## Tools & Techniques

Python, YOLOv8x, OpenCV, NumPy, multi-agent system design, computer vision, deep learning, object detection, data augmentation, edge AI, benchmarking, risk analysis

## Outcome

The system demonstrated how AI, computer vision, and intelligent agents can be combined to address real-world safety problems — highlighting applied skills in system design, applied machine learning, and data-driven decision-making for smart transport solutions.
