## Overview

This project designed and built a blockchain-based escrow system to automate property ownership transfers. Using Ethereum smart contracts and NFT-based digital property titles, the solution reduces reliance on intermediaries, improves transaction transparency, and strengthens data governance in the real estate transfer process.

## What I Did

- Designed a decentralised escrow architecture using **Ethereum smart contracts (Solidity)**
- Implemented **NFT-based digital property ownership** to represent and transfer real estate titles on-chain
- Built a **ReactJS + Web3.js** decentralised web application (dApp) integrated with MetaMask for user authentication and transaction signing
- Managed property and user data through **MongoDB**
- Tested the proof-of-concept using **Hardhat** and **Ganache** local blockchain environments
- Applied data governance principles to ensure security, transparency, and auditability throughout the transfer process

## Outcomes

- Solidity smart contracts for automated escrow and ownership transfer
- NFT-based digital property ownership model
- ReactJS + Web3.js decentralised web application
- MetaMask-integrated transaction workflow
- Hardhat/Ganache tested proof-of-concept

## Skills Used

Blockchain Solution Design, Data Governance, Smart Contract Development, Solidity, Decentralised Application (dApp) Development, Ethereum, NFT Implementation, ReactJS, Web3.js, MetaMask Integration, MongoDB, Hardhat, Ganache, System Testing
