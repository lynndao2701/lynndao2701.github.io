## Overview

This project built a machine learning model to predict corporate default risk using financial and firm-level variables. A Decision Tree classifier was the primary model, benchmarked against Logistic Regression, Random Forest, and Support Vector Classification (SVC). The project also analysed key drivers of default risk and proposed future improvements to increase robustness and real-world applicability.

## What I Did

- Selected and processed financial variables including leverage, liquidity, profitability, cash flow, ESG score, and industry classification
- Built and trained a **Decision Tree classifier** as the primary model
- Benchmarked performance against **Logistic Regression, Random Forest, and SVC**
- Applied **hyperparameter tuning via Grid Search** to optimise the Decision Tree model
- Conducted **correlation matrix and feature importance analysis** to identify key drivers of default risk
- Documented findings and produced a technical report with model evaluation metrics and business interpretation

## Outcomes

- Tuned Decision Tree classifier achieving up to **92% accuracy**
- Comparative evaluation across 4 ML models with performance metrics before and after tuning
- Correlation matrix and financial variable analysis
- Key finding: leverage, cash flow, and profitability are the strongest predictors of default risk
