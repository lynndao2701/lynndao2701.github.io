## Overview

Wildlife-vehicle collisions (WVCs) are a growing safety and conservation problem near forested roads. This project designed and implemented an AI-powered multi-agent system using YOLOv8x for real-time wildlife detection, with proactive driver alert capabilities to reduce collisions. The system supports transport safety, biodiversity conservation, and aligns with UN SDG 11 for sustainable communities.

## What I Did

- Designed a **multi-agent system architecture** with four specialised agents: Perception Agent, Threat Assessment Agent, Communication Agent, and Monitoring Agent
- Trained **YOLOv8x** for wildlife object detection, benchmarking it against YOLOv8n and Faster R-CNN
- Applied **data preprocessing and augmentation** to improve model robustness across varying environmental conditions
- Evaluated detection performance across day, night, rain, and fog conditions
- Conducted **risk analysis** and assessed the system's real-world deployment feasibility

## Outcomes

- Multi-agent system architecture with four coordinated agents
- High-accuracy YOLOv8x detection model
- Benchmarking results comparing YOLOv8x against YOLOv8n and Faster R-CNN
- Environmental condition evaluation results

## Skills Used

Python, YOLOv8x, OpenCV, NumPy, Computer Vision, Deep Learning, Object Detection, Multi-agent System Design, Data Preprocessing, Data Augmentation, Benchmarking, Risk Analysis, System Architecture
